🐍 Snake Water Gun Game

A simple and interactive Snake Water Gun game built using Python Flask, HTML, CSS, and JavaScript. The player selects a move, the computer generates a random move, and the game instantly displays the winner.

✨ Features
Interactive UI
Real-time gameplay
Random computer move generation
Winner determination logic
Flask backend API
JavaScript Fetch API integration
Responsive design
🛠️ Technologies Used
Python
Flask
HTML5
CSS3
JavaScript
